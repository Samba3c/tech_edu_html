教師指引

【課程設計理念】

這套遊戲集專為國小三年級學生設計，旨在透過遊戲結合英文及資訊技能，提升學生操作滑鼠、鍵盤輸入等基本資訊能力。所有遊戲都不需要網路連接，可獨立運行，並且不需要安裝任何額外軟體。

【系統環境需求】

  • 作業系統
    • Windows 11
  • 顯示設定
    • 螢幕解析度：1920 x 1080
    • 螢幕及瀏覽器縮放比例：100%
  • 注意事項
    • 為確保遊戲介面正常顯示，請務必依照建議設定螢幕解析度和縮放比例
    • 若顯示異常，請檢查並調整顯示設定

【遊戲詳細說明】

基礎練習遊戲（⭐）

  • 中文注音指法
    • 配合課程：注音輸入法教學
    • 使用時機：課堂練習或課後複習
    • 特色：真實鍵盤配置、輸入字元的顏色指引或錯誤提示，並給予1~7星的星等回饋
  • 英文字母指法
    • 配合課程：英文打字入門
    • 使用時機：英文課程輔助教材
    • 特色：提供10秒暖身、60秒挑戰、90秒練習模式，多種模式供教學或練習使用
  • 迷你小畫家
    • 配合課程：滑鼠操作入門
    • 使用時機：美術或資訊課程
    • 特色：一個簡單卻強大的繪圖工具，可以選擇畫筆形狀、畫筆大小、顏色挑選、油漆桶填色、清除畫布，並可返回5次動作。可以下載存檔，也可以上傳圖檔繼續繪圖。

【進階練習遊戲（⭐⭐）】

  • 貪食蛇遊戲
    • 配合課程：英文單字學習
    • 使用時機：英文課程活動
    • 特色：認識中英文對照，並能輸入正確的英文單字，按下Enter之後，啟動貪食蛇遊戲。透過鍵盤上下左右控制貪食蛇，依照英文單字的字母順序，逐一吃完英文字母。搶快過關，藉此練習英文拼字，也熟練操作鍵盤的上下左右。
  • 卡牌記憶遊戲
    • 配合課程：英文詞彙學習
    • 使用時機：課堂活動或課後練習
    • 特色：訓練滑鼠操作的靈敏度並考驗記憶力，以及英文識字能力，進行卡牌配對挑戰。採用emoji可愛圖文字，暢通網頁遊戲的進行。

【挑戰級遊戲（⭐⭐⭐）】

  • 塔防打怪遊戲
    • 配合課程：英文字母快速識別
    • 使用時機：課堂競賽活動
    • 特色：訓練快速反應和打字準確度，隨機出題的字母砲彈沒有做重複字母的篩選，運氣好的話可以同時發射2個砲彈，甚至同時發射3個。
  • 卡牌配對挑戰
    • 配合課程：英文單字理解
    • 使用時機：課堂評量活動
    • 特色：挑戰英文單字卡配對中文卡片，同時考驗滑鼠操作速度與準度，關卡時間愈來愈少，挑戰無限關卡！！

【專家級遊戲（⭐⭐⭐⭐）】

  • 射泡泡遊戲
    • 配合課程：英文打字速度訓練
    • 使用時機：綜合英文打字進階練習
    • 特色：鍵盤搶快輸入英文試著生存下來。隨機出題的泡泡沒有做重複字母的篩選，運氣好的話可以同時打破2個泡泡，甚至同時打破3個。
  • 侏儸紀逃亡
    • 配合課程：英文單字綜合運用
    • 使用時機：課程總結活動
    • 特色：恐龍逃出來了！快點完成英文單字，就會有水果加速逃亡者的奔跑速度，進到洞裡躲避恐龍。恐龍與人物的風格採用 ASCII 繪圖，維持獨有可愛的風格。

【建議使用順序】

  1. 從基礎練習遊戲開始，讓學生熟悉基本操作
  2. 待學生熟練後，進入進階練習遊戲
  3. 視學生程度，選擇性使用挑戰級遊戲
  4. 專家級遊戲建議作為獎勵或課後練習使用
